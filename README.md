# Hunter-Yonkou
Scanner offensive automatizado para web: enumera subdomínios (subfinder), coleta URLs (gau), valida hosts ativos e executa fuzzing inteligente de parâmetros com payloads mutados e bypass de WAF. Detecta XSS, SQLi, SSTI, SSRF, RCE e IDOR com análise de resposta, diff e heurísticas, usando alta concorrência.
